¿Como funciona?

1. Registrese en la app
2. Inicie sesión
3. Cargue audios (implementado, pero no se verifica que el archivo sea de audio)
4. Modifique atributos (proximamente)
5. Genere caracteristicas (proximamente)
6. Descargue resultados (solo implementada la vista)

Inicialmente guardar los resultados en el servidor era una opción, pero posiblemente no genera mucho valor para
el usuario, asi que para evitar complicaciones innecesarios, no se implementará dicha funcionalidad.

Para que funcione el proyecto:

ejecutar: python manage.py runserver
